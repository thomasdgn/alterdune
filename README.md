# alterdune
Ce projet propose une approche ludique et pédagogique à travers la création d’un mini-jeu de type RPG nommé ALTERDUNE. L’univers est particulièrement adapté à la POO : le joueur, les monstres, les objets, les actions et les combats peuvent être modélisés sous forme d’objets, organisés en classes et en hiérarchies simples.
